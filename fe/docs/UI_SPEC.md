# 게임 화면 UI 스펙 (FE 구현용)

모바일 세로 기준. 단위는 뷰포트 기준 % 또는 고정값 명시.

---

## 1. 전체 화면 구조 (세로 분할)

| 구역 | 높이(% of viewport) | 비고 |
|------|---------------------|------|
| 상단 네비게이션 바 | 5% | 고정 |
| 라이브 영상 영역 | 35% | 고정 |
| 플레이어 패널 + POT 영역 | 45% | 스크롤 가능 |
| 내 패 영역 | 10% | 고정 |
| 하단 네비게이션 바 | 5% | 고정 |

- **배경**: 어두운 톤 (검정/짙은 갈색), 카지노 테마. `back.png` 또는 그라데이션 사용.
- **글래스 효과**: 상단 오버레이 반투명 20~30% (선택).

---

## 2. 상단 네비게이션 바 (row 1)

- **Layout**: 단일 row, flex, `justify-content: space-between`, `align-items: center`.
- **높이**: 5vh (또는 min-height 40px).
- **배경**: 검정 또는 매우 짙은 회색, 약간 투명 가능.
- **좌우 패딩**: 8px ~ 12px.

| 요소 | 위치 | 내용 | 폰트 | 색상 | 아이콘 |
|------|------|------|------|------|--------|
| 로고 | 좌측 | "Zenith Park" 또는 로고 이미지 | 14~16pt, bold | 흰색 | 없음 (텍스트 또는 zp.png) |
| 게임 정보 | 중앙 또는 좌측 다음 | "10-2K 똑! 등1 - 114572998-11" | **11pt** | 흰색 | 없음 (또는 블라인드/룸 아이콘) |
| 우측 표시 | 우측 | "0" (알림/뷰어 수 등) | **11pt** | 흰색 | 없음 (선택: 벨/사람 아이콘) |

- **Col 비율**: 로고 고정폭, 게임정보 `flex: 1` 중앙, 우측 고정폭.

---

## 3. 라이브 영상 영역

- **Layout**: full width, 높이 35vh.
- **내용**: iframe 또는 video (HLS 스트림).
- **오버레이 (영상 우상단)**:
  - "LIVE" 뱃지: 작은 빨간 사각, 흰 글자 "LIVE", **11pt**.
  - 시청자 수: "1.8" + 사람 아이콘, **11pt**, 흰색.
- **테두리**: 없음 또는 0. border-radius 선택.

---

## 4. 플레이어 패널 + POT 영역 (스크롤)

- **Layout**: 세로 스크롤, 내부는 column. 높이 45vh.
- **배경**: 짙은 갈색/골드 그라데이션, 반투명.
- **패딩**: 8px ~ 12px.

### 4.1 플레이어 패널 (최대 9명, 4:4 배치)

- **Grid**: `grid-template-columns: repeat(4, 1fr)`, `gap: 4px`.
- **한 셀(한 명)**: 카드형 패널, rounded, 그림자.

**한 플레이어 패널 내부 (column, 3 rows):**

| Row | 내용 | 폰트 | 색상 | 아이콘 |
|-----|------|------|------|--------|
| Row 1 | 아이디 + (남은시간 · 배팅상태) | **9pt** (작게) | 흰색 | ✅ **사람 아이콘**(아이디), **시계 아이콘**(시간), **배팅/상태 아이콘**(상태) |
| Row 2 | 배팅 금액 (예: "1,000", "0") | **9pt** 또는 11pt | 흰색 | 없음 |
| Row 3 | 카드 2장 (썸네일) | - | - | ✅ 카드 이미지 2개 |

- **아바타**: 원형 외곽선, 내부 **사람 아이콘** (갈색/흰색). 크기: row 높이의 약 30~40% 또는 24~32px.
- **카드 썸네일**: 24×34px ~ 28×40px, object-fit: cover.
- **패널 배경**: 짙은 갈색, border-radius 6~8px, border 1px rgba(255,255,255,0.2).

### 4.2 POT 표시 (플레이어 목록 아래)

- **Layout**: 한 row, 가운데 정렬.
- **내용**: "POT 57,000".
- **폰트**: **18~20pt** (강조), 숫자 bold.
- **색상**: 흰색.
- **아이콘**: ✅ **POT/컵 아이콘** (선택).
- **배경**: 영역 배경과 동일 또는 구분선만.

---

## 5. 내 패 영역 ("내패")

- **Layout**: 한 row, `align-items: center`, `gap: 8px`.
- **높이**: 10vh (또는 min-height 56px).
- **배경**: 상단 영역과 동일 (짙은 갈색/골드).

| 요소 | 내용 | 폰트 | 색상 | 아이콘 |
|------|------|------|------|--------|
| 그리드 아이콘 | 좌측 | - | 흰색 | ✅ **그리드(카드) 아이콘** |
| 라벨 | "내패" | **11pt** | 흰색 | 없음 |
| 카드 1 | 카드 이미지 | - | - | ✅ 카드 이미지 |
| 카드 2 | 카드 이미지 | - | - | ✅ 카드 이미지 |

- **카드 크기**: 화면 너비의 약 **15%** each (또는 28×40px ~ 32×44px).
- **좌우 패딩**: 12px.

---

## 6. 하단 네비게이션 바

- **Layout**: 한 row, 2개 버튼 동일 비율 (`flex: 1`).
- **높이**: 5vh (또는 min-height 44px).
- **배경**: 짙은 갈색, solid.
- **gap**: 4~8px, 패딩 6~8px.

| 버튼 | 텍스트 | 폰트 | 색상 | 아이콘 |
|------|--------|------|------|--------|
| 좌측 | "게임이력" | **11pt** | 흰색 | 없음 (선택: 그리드 아이콘) |
| 우측 | "채팅" | **11pt** | 흰색 | 없음 (선택: 말풍선 아이콘) |

- **버튼 스타일**: Glass 또는 단색, border-radius 6~8px, border 2px (사용 시).

---

## 7. 게임이력 탭 (탭 선택 시 표시)

- **Layout**: **16×16 그리드**, `grid-template-columns: repeat(16, 1fr)`, `grid-template-rows: repeat(16, 1fr)`.
- **셀**: 정사각형에 가깝게, **border-radius: 2px**, **border: 2px** (흰색/반투명).
- **배치**: 컨테이너에 꽉 차게 (`width/height 100%`), `gap: 2px`.
- **배경**: 셀마다 rgba(0,0,0,0.2) 등.
- **폰트/아이콘**: 셀 내부에 패턴/숫자 넣을 경우 **9pt** 이하.

---

## 8. 채팅 탭

- **메시지 영역**: 스크롤, **글자 높이 작게** → **9pt**, line-height 1.25.
- **색상**: 흰색.
- **입력창 + 전송 버튼**: 11pt, 동일 스타일.

---

## 9. 칩 패널 (플레이어/POT 영역 내)

- **위치**: POT 아래 또는 내 패 위.
- **Layout**: 가로 스크롤 strip, **칩 이미지만** 사용 (직접 그린 버튼 없음).
- **칩 이미지**: 36×36px, `/assets/chips/chip_red.png` 등 값별 매핑.
- **아이콘**: 칩 이미지 자체가 아이콘 역할.

---

## 10. 액션 버튼 (2개)

- **위치**: 칩 패널 아래 또는 내 패 위.
- **버튼 1**: "2배 배팅".
- **버튼 2**: "배팅 취소".
- **Layout**: 한 row, 각 `flex: 1`, gap 6~8px.
- **폰트**: 11pt, 흰색.
- **아이콘**: 없음 (선택 가능).

---

## 11. 공통 값 요약

| 항목 | 값 |
|------|-----|
| 작은 글씨 (아이디, 시간, 상태, 채팅) | **9pt** |
| 일반 작은 글씨 (버튼, 라벨, 정보) | **11pt** |
| POT 금액 | **18~20pt**, bold |
| 기본 전경 색상 | 흰색 `#ffffff` |
| 배경 톤 | 검정/짙은 갈색 `#1a1a1a`, `#2d2d2d` 등 |
| 테두리 (패널) | 1~2px, rgba(255,255,255,0.2~0.3) |
| border-radius (패널/버튼) | 6~8px (일반), 2px (게임이력 셀) |
| 아이콘 크기 | 14×14px ~ 24×24px (문맥별) |

---

## 12. 아이콘 사용 체크리스트

| 위치 | 아이콘 | 용도 |
|------|--------|------|
| 상단 정보 | 블라인드/룸/베팅 (선택) | 10-2K, 룸, 베팅 0 |
| 플레이어 Row1 | 사람 | 아이디 |
| 플레이어 Row1 | 시계 | 남은시간 |
| 플레이어 Row1 | 배팅/상태 | 대기/배팅중/폴드 |
| POT | POT/컵 | POT 금액 앞 |
| 내 패 | 그리드/카드 | "내패" 라벨 앞 |
| 하단 탭 | 그리드, 말풍선 (선택) | 게임이력, 채팅 |

이 스펙으로 row/col, %, 폰트, 색상, 아이콘 여부를 일관되게 구현하면 됩니다.
